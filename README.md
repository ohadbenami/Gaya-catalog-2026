# כלי סקירת קטלוג GAYA — שכבת "מורה"

כלי פידבק ויזואלי מעל הקטלוג של GAYA. מאפשר לאוהד ולשגיא לסמן הערות, חיצים,
קווים וסימוני בלוקים ישירות מעל הקטלוג — כמו מורה שעובר על עבודה ומסמן הערות.

> **חשוב:** הכלי הזה **לא נוגע בקטלוג עצמו**. הקטלוג נטען כ-iframe ושכבת ההערות
> מצוירת מעליו. אין שום סיכון שהערות פנימיות ידלפו ללקוחות או לאנשי מכירות —
> הם פשוט לא מגיעים לדף הזה (הוא בריפו נפרד ומסומן `noindex`).

## קבצים
| קובץ | תפקיד |
|------|-------|
| `review.html` | מצב הסקירה — סרגל כלים + הקטלוג + שכבת ההערות |
| `review.js`   | מנוע הציור, השמירה, הייצוא/ייבוא |
| `index.html`  | דף נחיתה שמסביר את הכלי ומקשר ל-review |
| `feedback.json` | ההערות המשותפות (מתעדכן בייצוא והעלאה לריפו) |

## איך משתמשים
1. פותחים את `review.html` (או `index.html` → "פתח מצב סקירה").
2. בוחרים מי העורך (אוהד / שגיא) בסרגל למעלה.
3. בוחרים כלי: 📝 הערה · ➤ חץ · ／ קו · ▭ בלוק · 🖐️ ניווט.
4. מסמנים על הקטלוג. הכל נשמר אוטומטית בדפדפן (localStorage).

### קיצורי מקלדת
`1`/`V` ניווט · `2`/`N` הערה · `3`/`A` חץ · `4`/`L` קו · `5`/`R` בלוק · `Esc` חזרה לניווט.

## שיתוף הפידבק בין אוהד לשגיא
שמירה אוטומטית היא מקומית לדפדפן. כדי לשתף:
1. `⬇️ ייצוא` → יורד `feedback.json`.
2. מעלים אותו לריפו (מחליפים את הקיים).
3. הצד השני נכנס (נטען אוטומטית מ-`feedback.json`) או לוחץ `⬆️ ייבוא`.

הייבוא **ממזג** לפי מזהה — לא דורס הערות קיימות, רק מעדכן/מוסיף.

## הגדרת כתובת הקטלוג
ברירת המחדל מצביעה על `https://gaya-catalog-beige.vercel.app/gaya-catalog`.
אפשר לשנות בשדה "קטלוג" בסרגל. אם הקטלoג חוסם הטמעה ב-iframe (כותרת
`X-Frame-Options`), צריך לאפשר embedding לכתובת הזו, או להריץ את כלי הסקירה
מאותו דומיין כמו הקטלוג.

> **גישה:** כדי שאבנה את שכבת הסקירה *בתוך* הקוד של הקטלוג עצמו (ולא כשכבה מעל
> iframe), צריך לפתוח סשן Claude Code שמכוון לריפו של הקטלוג. הסשן הזה הוגבל
> לריפו `gaya-catalog-2026` בלבד, ולכן הכלי בנוי כאן כשכבה עצמאית שלא תלויה
> בקוד של שגיא.
